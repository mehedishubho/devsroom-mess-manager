# Plan 3.2: Advance balance + due balance + carry-forward

## Objective

Add the `due_balance` column to `advance_balances` (D-08), wire `Payment` writes for `advance_deposit` to immediately bump the member's `advance_balance` (D-07, D-10), expose a manager view of all members' advance/due balances (`/mess/advance-balances`), and a member view of their own (`/my` → "Advance balance" tab). The actual cross-month carry-forward math (D-09) is the month-close service's job and ships in Plan 3.4; this plan builds the **data model + the deposit hook** so the close service has a clean API to call later.

By the end of this plan: `advance_balances` has both `advance_balance` (credit) and `due_balance` (debt) columns; creating a payment with `type=advance_deposit` updates the member's row (idempotent via `firstOrCreate` + `update`); a manager can see all members' balances and manually adjust (with a reason); a member sees their own balance in `/my`. Every balance-changing write goes through the service so the audit trail is consistent.

**Requirements covered:** ADV-01, ADV-02, ADV-03, ADV-06, ADV-07, PAY-03 (advance_deposit semantics — schema and write path only; close-time carry-forward is in Plan 3.4).

### Task 4: Add `AdvanceBalanceService` (deposit, repay, adjust, carryForward)

**File:** `app/Services/AdvanceBalanceService.php` (new)

```php
<?php

namespace App\Services;

use App\Models\AdvanceBalance;
use App\Models\Mess;
use App\Models\Payment;
use App\Support\PaymentType;
use Illuminate\Support\Facades\DB;
use RuntimeException;

class AdvanceBalanceService
{
    /**
     * Apply the impact of a Payment to the member's advance/due balance.
     * Only `advance_deposit` touches `balance` (D-07). `bill_payment` is a no-op here.
     */
    public function applyPayment(Payment $payment): void
    {
        if ($payment->type !== PaymentType::ADVANCE_DEPOSIT) {
            return;
        }

        DB::transaction(function () use ($payment) {
            $row = AdvanceBalance::query()
                ->where('member_id', $payment->member_id)
                ->lockForUpdate()
                ->first();

            if (! $row) {
                $row = AdvanceBalance::create([
                    'mess_id' => Mess::activeId(),
                    'member_id' => $payment->member_id,
                    'balance' => 0,
                    'due_balance' => 0,
                    'last_updated_at' => now(),
                ]);
            }

            $row->balance = (float) $row->balance + (float) $payment->amount;
            $row->last_updated_at = now();
            $row->save();
        });
    }

    /**
     * Manager-side manual adjustment with a reason (D-07 / D-11).
     * Positive amount increases advance (credit); negative amount increases due (debt).
     * Records the reason in the notes — it is **also** captured via Payment audit for advance_deposit.
     */
    public function adjust(int $memberId, float $amount, string $reason, int $enteredBy): AdvanceBalance
    {
        if ($amount === 0.0) {
            throw new RuntimeException('Adjustment amount cannot be zero.');
        }

        return DB::transaction(function () use ($memberId, $amount, $reason, $enteredBy) {
            $row = AdvanceBalance::query()
                ->where('member_id', $memberId)
                ->lockForUpdate()
                ->firstOrCreate(
                    ['mess_id' => Mess::activeId(), 'member_id' => $memberId],
                    ['balance' => 0, 'due_balance' => 0, 'last_updated_at' => now()]
                );

            if ($amount > 0) {
                $row->balance = (float) $row->balance + $amount;
            } else {
                $row->due_balance = (float) $row->due_balance + abs($amount);
            }
            $row->last_updated_at = now();
            $row->save();

            \Illuminate\Support\Facades\Log::info('manual_balance_adjustment', [
                'member_id' => $memberId,
                'amount' => $amount,
                'reason' => $reason,
                'entered_by' => $enteredBy,
                'new_balance' => $row->balance,
                'new_due_balance' => $row->due_balance,
            ]);

            return $row;
        });
    }

    /**
     * Used by the month-close service in Plan 3.4.
     * $amount > 0 → credit to advance (member had excess from last month)
     * $amount < 0 → debit to due (member owed from last month)
     * $amount === 0 → no-op.
     */
    public function carryForward(int $memberId, float $amount): AdvanceBalance
    {
        return DB::transaction(function () use ($memberId, $amount) {
            $row = AdvanceBalance::query()
                ->where('member_id', $memberId)
                ->lockForUpdate()
                ->firstOrCreate(
                    ['mess_id' => Mess::activeId(), 'member_id' => $memberId],
                    ['balance' => 0, 'due_balance' => 0, 'last_updated_at' => now()]
                );

            if ($amount > 0) {
                $row->balance = (float) $row->balance + $amount;
            } elseif ($amount < 0) {
                $row->due_balance = (float) $row->due_balance + abs($amount);
            }
            $row->last_updated_at = now();
            $row->save();

            return $row;
        });
    }
}
```

### Task 5: Wire `PaymentService::create()` to call `AdvanceBalanceService::applyPayment()`

Inject `AdvanceBalanceService` and call it after the payment is created.

```php
<?php

namespace App\Services;

use App\Models\Mess;
use App\Models\Payment;
use App\Support\PaymentType;
use Illuminate\Contracts\Pagination\LengthAwarePaginator;
use Illuminate\Http\Request;

class PaymentService
{
    public function __construct(private readonly AdvanceBalanceService $balances) {}

    public function list(Request $request): LengthAwarePaginator
    {
        // unchanged from Plan 3.1
        $query = Payment::query()->with(['member', 'enteredBy'])->latest('date');
        if ($memberId = $request->query('member_id')) {
            $query->where('member_id', (int) $memberId);
        }
        if ($method = $request->query('method')) {
            $query->where('method', $method);
        }
        if ($from = $request->query('from')) {
            $query->whereDate('date', '>=', $from);
        }
        if ($to = $request->query('to')) {
            $query->whereDate('date', '<=', $to);
        }
        return $query->paginate(50)->withQueryString();
    }

    public function listForMember(int $memberId, int $perPage = 30): LengthAwarePaginator
    {
        return Payment::query()->where('member_id', $memberId)->with('enteredBy')
            ->latest('date')->latest('id')->paginate($perPage);
    }

    public function create(array $data): Payment
    {
        $payment = Payment::create([
            'mess_id' => Mess::activeId(),
            'member_id' => $data['member_id'],
            'date' => $data['date'],
            'amount' => $data['amount'],
            'method' => $data['method'],
            'reference' => $data['reference'] ?? null,
            'notes' => $data['notes'] ?? null,
            'type' => $data['type'] ?? PaymentType::BILL_PAYMENT,
            'entered_by' => auth()->id(),
        ]);

        $this->balances->applyPayment($payment);

        return $payment;
    }

    public function update(Payment $payment, array $data): Payment
    {
        $payment->update([
            'member_id' => $data['member_id'],
            'date' => $data['date'],
            'amount' => $data['amount'],
            'method' => $data['method'],
            'reference' => $data['reference'] ?? null,
            'notes' => $data['notes'] ?? null,
            'type' => $data['type'] ?? $payment->type,
        ]);

        return $payment->refresh();
    }
}
```

```php
<?php

namespace App\Http\Controllers\Mess;

use App\Http\Controllers\Controller;
use App\Http\Requests\Mess\AdjustAdvanceBalanceRequest;
use App\Models\Member;
use App\Services\AdvanceBalanceService;
use App\Support\MemberStatus;
use Illuminate\Http\RedirectResponse;
use Illuminate\View\View;

class AdvanceBalanceController extends Controller
{
    public function __construct(private readonly AdvanceBalanceService $service) {}

    public function index(): View
    {
        $members = Member::query()
            ->where('status', MemberStatus::ACTIVE)
            ->with('advanceBalance')
            ->orderBy('name')
            ->get();

        return view('mess.advance-balances.index', compact('members'));
    }

    public function adjust(Member $member): View
    {
        $member->load('advanceBalance');

        return view('mess.advance-balances.adjust', compact('member'));
    }

    public function storeAdjust(AdjustAdvanceBalanceRequest $request, Member $member): RedirectResponse
    {
        $this->service->adjust(
            (int) $member->id,
            (float) $request->validated('amount'),
            (string) $request->validated('reason'),
            (int) $request->user()->id,
        );

        return redirect()
            ->route('mess.advance-balances.index')
            ->with('success', __('Balance adjusted for :name.', ['name' => $member->name]));
    }
}
```

```php
<?php

namespace Tests\Feature\Mess;

use App\Models\AdvanceBalance;
use App\Models\Member;
use App\Models\Mess;
use App\Models\Payment;
use App\Models\User;
use App\Support\MemberStatus;
use App\Support\PaymentMethod;
use App\Support\PaymentType;
use HasinHayder\Tyro\Models\Role;
use Illuminate\Foundation\Testing\RefreshDatabase;
use Tests\TestCase;

class AdvanceBalanceTest extends TestCase
{
    use RefreshDatabase;

    protected function setUp(): void
    {
        parent::setUp();
        $this->seedTyroRoles();
        $mess = Mess::factory()->create();
        config(['mess.active_mess_id' => $mess->id]);
    }

    public function test_advance_deposit_credits_member_balance(): void
    {
        $member = Member::factory()->create(['status' => MemberStatus::ACTIVE]);
        Payment::factory()->create([
            'member_id' => $member->id,
            'type' => PaymentType::ADVANCE_DEPOSIT,
            'amount' => 2000,
            'method' => PaymentMethod::BKASH,
            'reference' => 'ADV-1',
        ]);

        $row = AdvanceBalance::where('member_id', $member->id)->first();
        $this->assertNotNull($row);
        $this->assertEquals(2000.00, (float) $row->balance);
        $this->assertEquals(0.00, (float) $row->due_balance);
    }

    public function test_bill_payment_does_not_touch_balance(): void
    {
        $member = Member::factory()->create(['status' => MemberStatus::ACTIVE]);
        Payment::factory()->create([
            'member_id' => $member->id,
            'type' => PaymentType::BILL_PAYMENT,
            'amount' => 1500,
        ]);

        $this->assertDatabaseMissing('advance_balances', ['member_id' => $member->id]);
    }

    public function test_multiple_deposits_accumulate(): void
    {
        $member = Member::factory()->create(['status' => MemberStatus::ACTIVE]);
        Payment::factory()->create(['member_id' => $member->id, 'type' => PaymentType::ADVANCE_DEPOSIT, 'amount' => 1000]);
        Payment::factory()->create(['member_id' => $member->id, 'type' => PaymentType::ADVANCE_DEPOSIT, 'amount' => 500]);

        $row = AdvanceBalance::where('member_id', $member->id)->first();
        $this->assertEquals(1500.00, (float) $row->balance);
    }

    public function test_admin_can_adjust_balance_with_positive_amount(): void
    {
        $admin = User::factory()->create();
        $admin->assignRole(Role::where('slug', 'admin')->first());
        $member = Member::factory()->create(['status' => MemberStatus::ACTIVE]);

        $this->actingAs($admin)
            ->post(route('mess.advance-balances.store-adjust', $member), [
                'amount' => 500,
                'reason' => 'Manual credit',
            ])
            ->assertRedirect(route('mess.advance-balances.index'));

        $row = AdvanceBalance::where('member_id', $member->id)->first();
        $this->assertEquals(500.00, (float) $row->balance);
    }

    public function test_admin_can_adjust_balance_with_negative_amount(): void
    {
        $admin = User::factory()->create();
        $admin->assignRole(Role::where('slug', 'admin')->first());
        $member = Member::factory()->create(['status' => MemberStatus::ACTIVE]);

        $this->actingAs($admin)
            ->post(route('mess.advance-balances.store-adjust', $member), [
                'amount' => -300,
                'reason' => 'Manual due',
            ])
            ->assertRedirect(route('mess.advance-balances.index'));

        $row = AdvanceBalance::where('member_id', $member->id)->first();
```
