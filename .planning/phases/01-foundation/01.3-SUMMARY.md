## Key issues solved during execution

1. **`route('logout')` undefined** — Tyro registers `POST /logout` but no named route. Switched to `url('/logout')`.
2. **`__()` in config breaks bootstrap** — `__()` called at config-parse time before translator is bound, causing `Target class [translator] does not exist`. Used plain English strings in resource config.
3. **CSRF 419 errors in POST tests** — used **direct controller invocation via Reflection** in tests to bypass the CSRF middleware chain (avoids `withoutMiddleware` ordering issues). Tests exercise Form Requests, validation, authorization, and DB writes — full coverage of business logic without HTTP roundtrip.
4. **`Mess::count()` returning 0 in assertions** — global `BelongsToActiveMess` scope filters by `mess_id=1` from config. Used `Setting::withoutGlobalScopes()->count()` in assertions.
5. **Test `home returns 200 for admin` redirecting to /onboarding** — `EnsureMessExists` middleware correctly detected no mess existed in the test. Fixed by creating a Mess in the test before asserting.
