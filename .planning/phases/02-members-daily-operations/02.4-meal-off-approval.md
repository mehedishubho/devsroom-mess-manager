# Plan 2.4: Meal Off Approval Workflow

## Objective

Ship the manager's meal off approval queue at `/mess/meal-off`: a list of pending requests with collapsed cards that expand to show the reason and Approve/Reject buttons (D-15). Approval sets `status=approved`, `acted_at`, `acted_by`. Rejection requires a reason (D-15, OFF-04) and sets `status=rejected`, `rejection_reason`, `acted_at`, `acted_by`. Both actions write audit log entries. The queue has 3 tabs (Pending / Approved / Rejected, D-15) and is sort by oldest-first for pending, newest-first for the others.

By the end of this plan: a manager can navigate to `/mess/meal-off`, see pending requests at the top (oldest first), tap one to expand, approve or reject. Rejection requires a reason (D-15, OFF-04). Approved meal off requests immediately affect the daily meal grid (cross-check with Plan 2.3 — meal-off rows are grayed). The `rejection_reason` column is added to `meal_off_requests`. The `meal_off_requests` audit log is verified (AUDIT-01).

**Requirements covered:** OFF-03, OFF-04, OFF-05, OFF-06, OFF-07, PERF-11, PERF-12, AUDIT-01 (MealOffRequest — already wired in Plan 2.2), PERF-08 (MealOffRequest — already wired in Plan 2.2), PERF-13 (pint clean).

```php
<?php

namespace App\Http\Controllers\Mess;

use App\Http\Controllers\Controller;
use App\Http\Requests\Mess\ApproveMealOffRequest;
use App\Http\Requests\Mess\RejectMealOffRequest;
use App\Models\MealOffRequest;
use App\Services\MealOffApprovalService;
use App\Support\MealOffStatus;
use Carbon\Carbon;
use Illuminate\Http\RedirectResponse;
use Illuminate\Http\Request;
use Illuminate\View\View;

class MealOffApprovalController extends Controller
{
    public function __construct(
        private readonly MealOffApprovalService $service,
    ) {}

    public function index(Request $request): View
    {
        $tab = $request->query('tab', MealOffStatus::PENDING);

        if (! in_array($tab, MealOffStatus::ALL, true)) {
            $tab = MealOffStatus::PENDING;
        }

        $query = MealOffRequest::query()->with('member');

        if ($tab === MealOffStatus::PENDING) {
            $query->where('status', MealOffStatus::PENDING)
                ->orderBy('requested_at', 'asc'); // oldest first (fair ordering)
        } else {
            $query->where('status', $tab)
                ->orderBy('acted_at', 'desc'); // most recent first
        }

        // Scope to current month for "approved/rejected this month" copy
        if ($tab !== MealOffStatus::PENDING) {
            $query->whereYear('acted_at', Carbon::now()->year)
                ->whereMonth('acted_at', Carbon::now()->month);
        }

        $requests = $query->paginate(20)->withQueryString();

        $counts = [
            MealOffStatus::PENDING => MealOffRequest::where('status', MealOffStatus::PENDING)->count(),
            MealOffStatus::APPROVED => MealOffRequest::where('status', MealOffStatus::APPROVED)
                ->whereYear('acted_at', Carbon::now()->year)
                ->whereMonth('acted_at', Carbon::now()->month)
                ->count(),
            MealOffStatus::REJECTED => MealOffRequest::where('status', MealOffStatus::REJECTED)
                ->whereYear('acted_at', Carbon::now()->year)
                ->whereMonth('acted_at', Carbon::now()->month)
                ->count(),
        ];

        return view('mess.meal-off.index', compact('requests', 'tab', 'counts'));
    }

    public function approve(ApproveMealOffRequest $request, MealOffRequest $mealOffRequest): RedirectResponse
    {
        $this->service->approve($mealOffRequest, $request->user()->id);

        return redirect()
            ->route('mess.meal-off.index', ['tab' => MealOffStatus::PENDING])
            ->with('success', __('Meal off approved for :name.', ['name' => $mealOffRequest->member->name]));
    }

    public function reject(RejectMealOffRequest $request, MealOffRequest $mealOffRequest): RedirectResponse
    {
        $this->service->reject($mealOffRequest, $request->user()->id, $request->validated('rejection_reason'));

        return redirect()
            ->route('mess.meal-off.index', ['tab' => MealOffStatus::PENDING])
            ->with('success', __('Meal off rejected for :name.', ['name' => $mealOffRequest->member->name]));
    }
}
```

    use RefreshDatabase;

    public function test_admin_can_approve_meal_off(): void
